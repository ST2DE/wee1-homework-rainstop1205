# 如何使用Postman測試HTTP（Windows 版本）

## 什麼是Postman？
> **免費**的 Chrome 擴充元件(extensions)<br>

> GUI(Graphical User Interface)介面

> 用來測試 REST(Representational State Transfer，一種軟體架構的設計概念，HTTP 即為一項實作) 服務應用正常與否

## Postman 使用方法
1. 前往[這裡](https://chrome.google.com/webstore/detail/postman-interceptor/aicmkgpgakddgnaphhhpliifpcfhicfo?hl=zh-TWl=zh-TW)下載該擴充元件
2. 在 Chrome 瀏覽器輸入```chrome://apps/```開啟應用程式頁面，點選 Postman 開啟程式
3. 依個人需求測試不同的 Request Method (GET、POST...)
